# FloodGuard Admin Web Landing Page

React Native Web landing page for the Flood Monitoring System.

## Location
This folder is located at: `frontend/admin/web-admin/`

## Setup

1. Navigate to this folder:
```bash
cd frontend/admin/web-admin
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

Or:
```bash
npm start
# Then press 'w' to open in web browser
```

## Project Structure

- `App.js` - Main landing page component with all sections
- `package.json` - Dependencies and scripts
- `app.json` - Expo configuration (web-only)
- `babel.config.js` - Babel configuration

## Features

- Navigation bar with smooth scrolling
- Hero section with CTA buttons
- Alerts section with image
- Features grid
- Prevention guide
- Statistics section
- Admin preview section
- Footer with links
